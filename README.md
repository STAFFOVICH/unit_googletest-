Примечания:
В тестах использованы EXPECT_THROW с std::out_of_range — можно заменить на любой другой тип исключения, если в реализации используется другой.
Метод pop в BinaryTree принимает значение для удаления (в отличие от очереди и кучи, где удаляется первый/наименьший элемент).
Все классы шаблонные — тесты написаны для int, но можно параметризовать тесты через TYPED_TEST, если нужно проверять разные типы.
